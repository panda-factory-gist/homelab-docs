# Bienvenue sur mon Homelab DevOps

Documentation complète de mon infrastructure d'apprentissage.

## Architecture Générale

- **Hyperviseur** : Proxmox VE 8.x
- **Routeur/Firewall** : OPNsense 24.7
- **Segmentation** : 4 réseaux isolés (LAN, DMZ, DEV, MGMT)

## Objectifs

- Apprendre le DevOps
- Préparer les certifications (AWS, Azure, CCNA ...)
- Expérimenter Ansible, Kubernetes, CI/CD, Docker ...
